什么是 Node？

Node 是一个 JS 的允许环境。

拥有比浏览器更多的能力。

> JavaScript 一般指在浏览器运行的 JS，NodeJS 指在 Node 环境运行的 JS!

## NodeJS 可以干什么

NodeJS 是单线程异步的程序，IO 处理速度是非常快的。

IO：input/output 输入输出。

Node 是适合做一些简单的 API 请求，不适合做一些复杂的请求。
